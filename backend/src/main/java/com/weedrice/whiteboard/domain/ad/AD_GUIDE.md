# Ad 도메인 가이드

`ad` 도메인은 노출 위치별 광고 조회와 클릭 로그 적재를 담당합니다.

## 1. 주요 기능 및 로직
- 광고 조회: 활성 상태이면서 노출 기간 내인 광고를 위치별로 조회하고 노출 수를 증가시킵니다.
- 클릭 기록: 클릭 시 광고 클릭 수를 증가시키고, 로그인 사용자 정보와 함께 클릭 로그를 남긴 뒤 타겟 URL을 반환합니다.
- 사용자 매핑: 인증 정보가 있으면 클릭 로그에 사용자 FK를 저장해 중복/부정 사용 분석에 활용합니다.

## 2. API Endpoints

| Method | URI | 설명 |
| :----- | :------------------------ | :------------------ |
| `GET` | `/api/v1/ads?placement=` | 위치별 광고 조회 |
| `POST` | `/api/v1/ads/{adId}/click` | 광고 클릭 기록 및 이동 URL 응답 |

## 3. 관련 DB 테이블

| 테이블명 | 엔티티 | 설명 |
| :------- | :----- | :--- |
| `ads` | `Ad` | 광고 메타 정보와 노출/클릭 집계 |
| `ad_click_logs` | `AdClickLog` | 광고 클릭 로그(사용자/시간/IP) |
