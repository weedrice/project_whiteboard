# Tag 도메인 가이드

`tag` 도메인은 게시글 태그 부착/제거와 인기 태그 조회를 담당합니다.

## 1. 주요 기능 및 로직
- 태그 처리: 게시글 저장/수정 시 기존 태그를 비교하여 삭제하거나 신규 태그를 생성/연결하고 사용량(post_count)을 갱신합니다.
- 인기 태그: 사용량 상위 태그를 조회해 제공합니다.

## 2. API Endpoints

| Method | URI | 설명 |
| :----- | :---------------------------------- | :--------------------------- |
| `GET` | `/api/v1/tags` | 인기 태그 목록 |
| `GET` | `/api/v1/tags/{tagId}/posts` | 태그가 달린 게시글 목록 |

## 3. 관련 DB 테이블

| 테이블명 | 엔티티 | 설명 |
| :------- | :----- | :--- |
| `tags` | `Tag` | 태그 이름/사용량 |
| `post_tags` | `PostTag` | 게시글-태그 매핑 |
