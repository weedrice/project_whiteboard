# Sanction 도메인 가이드

`sanction` 도메인은 사용자 제재(정지/경고) 기록과 조회를 관리합니다.

## 1. 주요 기능 및 로직
- 제재 생성: 슈퍼관리자 권한으로 대상 사용자에 제재 유형(BAN 등)과 기간/비고를 등록하며, BAN 시 즉시 계정 상태를 정지합니다.
- 제재 조회: 전체 또는 특정 사용자의 제재 이력을 페이지 단위로 조회합니다.
- 차단 여부 확인: 현재 시점에 유효한 BAN 제재가 있는지 확인하는 유틸 제공.

## 2. API Endpoints

| Method | URI | 설명 |
| :----- | :------------------------------------ | :--------------------- |
| `POST` | `/api/v1/admin/sanctions` | 사용자 제재 등록 |
| `GET` | `/api/v1/admin/sanctions` | 제재 이력 조회(필터 지원) |

## 3. 관련 DB 테이블

| 테이블명 | 엔티티 | 설명 |
| :------- | :----- | :--- |
| `sanctions` | `Sanction` | 제재 유형/기간/관리자/대상 사용자 |
