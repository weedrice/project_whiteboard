# Point 도메인 가이드

`point` 도메인은 사용자 포인트 잔액과 적립/차감 이력을 관리합니다.

## 1. 주요 기능 및 로직
- 포인트 조회: 사용자별 현재 포인트를 반환합니다.
- 이력 조회: 적립/차감 유형별 포인트 변동 내역을 페이지 단위로 조회합니다.
- 포인트 적립: 게시글/댓글 등 액션 수행 시 적립하고 잔액·이력을 함께 저장합니다.
- 포인트 차감: 강제 차감 시 잔액을 감소시키고 이력에 PENALTY 유형으로 기록합니다.

## 2. API Endpoints

| Method | URI | 설명 |
| :----- | :-------------------------------- | :---------------------- |
| `GET` | `/api/v1/points/me` | 내 포인트 조회 |
| `GET` | `/api/v1/points/me/history` | 내 포인트 변동 이력 조회 |

## 3. 관련 DB 테이블

| 테이블명 | 엔티티 | 설명 |
| :------- | :----- | :--- |
| `user_points` | `UserPoint` | 사용자별 포인트 잔액 |
| `point_histories` | `PointHistory` | 적립/차감 이력 및 잔액 스냅샷 |
